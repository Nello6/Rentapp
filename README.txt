# aVerySADProject- Rentapp
RentApp: è un progetto universitario nell'ambito Software and Architecture Design che permette la gestione di autonoleggi ed un sistema di booking prenotazioni per quest'ultimi.

## Installation

Richiede l'installazione di Tomcat e Microsoft SQL Server 2019.
Si rimanda alla documentazione del software per maggiori dettagli

## Usage

Per avviare correttamente l'applicazione:
1.Caricare il database su microsoft SQL Server 2019
2.Aprire il file .war con winrar, cercare e modificare l'application.propreties inserendo l'url dell' SQL Server.
3.Aprire Tomcat, caricare il file.war e avviare il server.
4.Il sito sarà visibile all'indirizzo localhost:8080/rentapp

Si rimanda alla documentazione del software per maggiori dettagli
